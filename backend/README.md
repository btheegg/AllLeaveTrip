# All Leave Trip Recommend System

협업 필터링을 이용해서, 창업 아이템 All Leave Trip 추천 시스템을 구현한 레포지터리입니다.

### 코스 추천받기

- 요청방식 : get
- 기능 : 게시물들의 목록을 보여준다.
- url : http://127.0.0.1:8000/course/
- 예시

#### Query Parameter

| 파라미터 | 필수여부 | 설명                                           |
| -------- | -------- | ---------------------------------------------- |
| day      | Y        | 며칠 여행할 것 인지를 나타내는 파라미터입니다. |
| userId   | Y        | 조회 할 유저의 ID입니다.                       |
| areaCode | Y        | 지역 코드입니다.                               |

#### areaCode

| areaCode | 지역 |
| :------: | ---- |
|    1     | 서울 |
|    2     | 부산 |
|    3     | 제주 |
